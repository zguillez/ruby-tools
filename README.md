# Getting Started

### Install gem

    gem install ruby-tools

# Add to gemfile:

	gem 'ruby-tools', '~> 0.0.1'

# Usage:

	#!/usr/bin/ruby
    
    require 'ruby-tools'
    
    RubyTools.version()
